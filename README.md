# Diwali Sales Data Analysis Project

## Project Summary

This project focuses on analyzing Diwali sales data to uncover customer purchasing behavior and sales trends. The objective was to clean and preprocess raw data, perform exploratory data analysis (EDA), and derive actionable business insights to improve customer targeting, inventory planning, and overall sales performance. Using Python and popular data analysis libraries, key customer segments and top-performing product categories were identified to support data-driven decision-making.

---

## Problem Statement

Businesses experience a surge in sales during festive seasons like Diwali. However, without proper data analysis, it becomes difficult to understand customer preferences, demand patterns, and revenue-driving segments. This project aims to analyze Diwali sales data to help businesses:

* Identify high-value customers
* Understand demographic-based purchasing patterns
* Optimize inventory and marketing strategies

---

## Tools & Technologies Used

* **Programming Language:** Python
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
* **Environment:** Jupyter Notebook

---

## Key Steps Performed

* Data cleaning and preprocessing (handling null values, data type corrections)
* Exploratory Data Analysis (EDA) using visualizations
* Customer segmentation based on gender, age group, occupation, and state
* Sales analysis to identify top-performing product categories and products

---

## Key Insights

* Identified potential customers across different states, age groups, and occupations
* Discovered gender-based and age-based purchasing trends
* Found top-selling product categories contributing maximum revenue
* Provided insights that can help improve inventory planning and customer experience

---

## Business Impact

* Enhanced customer targeting strategies
* Improved sales forecasting and inventory management
* Enabled data-driven decision-making for festive sales campaigns
